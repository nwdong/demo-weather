This is a demo system to display real time weather summary of capital cities in Australia.

Overall, this system is based on RESTful architecture:

  • Frontend, AngularJS framework (Angular-seed for bootstrap and Protractor for end to end testing).
  • Backend, Spring framework (Spring Boot for bootstrap and Spring Test for testing).
  • Integration, RESTful APIs of api.openweathermap.org for real time weather information.

Related code can be downloaded from https://github.com/nwdong/weather
